# Webpack Starter
Webpack starter kit

## Installation

```
npm install
```

## NPM Scripts
1. Run dev ```npm run dev```
2. Run build ```npm run build```

## WIP: New NPM Scripts

### Run static project
1. Run dev ```npm run dev:static```
2. Run build ```npm run build:static```

### Run react project
1. Run dev ```npm run dev:react```
2. Run build ```npm run build:react```
